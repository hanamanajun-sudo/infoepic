유튜브에서 영상 봤던 기록 지우고 싶을 때 혹은 검색했던 기록을 지우거나 정리하고 싶을 때 활용하기 좋습니다

모르면 헤매지만

알고 나면 간단한 방법이에요. 몰라서 검색해보실 분들을 위해 공유해요. 

(유튜브에 검색한 내용도 다 남는데 이것도 삭제 가능합니다.)

* * *

## **유튜브 영상** **시청 기록 일괄** **삭제 방법**

**1\. 유튜브 pc화면에서 좌측 상단에 햄버거 아이콘 클릭**

![유튜브 시청/검색 기록 삭제 방법(3초 해결) 이미지 1](/images/유튜브-시청_검색-기록-삭제-방법(3초-해결)/img.png)

**2\. 왼쪽 사이드바 중단에 시청기록 메뉴 클릭**

![유튜브 시청/검색 기록 삭제 방법(3초 해결) 이미지 2](/images/유튜브-시청_검색-기록-삭제-방법(3초-해결)/img_1.png)

**3\. 오른쪽에 시청기록 관련 메뉴 나오는데 여기서 시청기록 지우기 누르면 전체 시청기록 일괄 삭제**

![유튜브 시청/검색 기록 삭제 방법(3초 해결) 이미지 3](/images/유튜브-시청_검색-기록-삭제-방법(3초-해결)/img_2.png)

**4\. 마지막으로 삭제하기전 확인** 

본인 시청 기록이 중요하지 하다면 모르겠지만 개인적으로는 그리 지장은 없었어요.

다시 알고리즘 찾아주니 지워야 할 순간이라면 깔끔하게 지워보세요.

![유튜브 시청/검색 기록 삭제 방법(3초 해결) 이미지 4](/images/유튜브-시청_검색-기록-삭제-방법(3초-해결)/img_3.png)

* * *

## **유튜브 시청기록 개별 삭제, 검색 기록 삭제**

**1\. 위의 2번까지는 동일하고 이후에 오른쪽 아래 '시청기록 및 검색기록' 메뉴 클릭**

![유튜브 시청/검색 기록 삭제 방법(3초 해결) 이미지 5](/images/유튜브-시청_검색-기록-삭제-방법(3초-해결)/img_4.png)

2\. 유튜브 기록 페이지로 이동

아래 이미지에서 **1번**을 누르면 시청기록, 검색기록 기간별 삭제 가능

**2번** 누르면 해당일 삭제 가능 

**3번** 누르면 개별 시청기록, 검색 기록 삭제 가능

![유튜브 시청/검색 기록 삭제 방법(3초 해결) 이미지 6](/images/유튜브-시청_검색-기록-삭제-방법(3초-해결)/img_5.png)
