트렐로는 워낙 사용하기 편해서 그동안 사용하는데 불편함 없이 사용해 왔는데

쌓아 놨던 카드들을 정리하려고 보니까 마우스로 만 하기에는 불편함이 느껴져서

처음으로 단축키를 찾아보았어요.

단축키 사용하니까 몇배는 편해 졌어요. 일단 대충 필요한 것 만 외워 두기는 했는데

나중에 다시 찾아 볼 수 있게 정리했어요.

필요한 분 계시다면 도움 되셨으면 합니다. 

## **트렐로 데스크 탑에서 사용할 때에 키보드 단축키 정리** 

---

### **■새 카드 만들기**

> **N**

"n"을 누르면 현재 선택된 카드 다음 또는 빈 목록에 카드를 추가 할 수있는 팝업이 열립니다.

### ****■**\[카드 보관(아카이브, 사실상 삭제)하기\] - 트렐로 단축키**

> **C**

"c"를 누르면 카드가 보관됩니다.

> \*Tip. 트렐로 보관된 카드 어디에 있어? More> Archived items 에서 확인 가능

### ****■**\[카드 선택\] - 트렐로 단축키**

> ←  ↓ or J  ↑ or  K  →

화살표 키를 누르면 보드에서 가까운 카드를 선택합니다. 

"j"를 누르면 현재 카드 아래의 카드가 선택됩니다. 

"k"를 누르면 현재 카드 위의 카드가 선택됩니다.

### ****■**카드 열기 단축키**

> **엔터키**

"enter"를 누르면 현재 선택된 카드가 열립니다. 

### ****■**\[헤더 보드 메뉴 열기\] - 트렐로 단축키**

> B

"b"를 누르면 헤더에 보드 메뉴가 열립니다. 위쪽 및 아래쪽 화살표를 사용하여

보드를 검색하고 보드를 탐색 할 수 있습니다. 

보드를 선택한 상태에서 Enter 키를 누르면 열립니다.

### ****■**제목 수정 하기**

> **T**

카드를 볼 때 "t"를 누르면 제목이 편집됩니다. 

카드 위로 마우스를 가져간 경우 "t"를 누르면 카드가 열리고 제목 편집이 가능

### ****■**카드 복사/ 붙여 넣기**

카드 위로 마우스를 가져 가면 Control + C (Windows) 또는 Command + C (Mac)를 누르면

카드가 클립 보드에 복사됩니다. 

목록 위로 마우스를 가져 가면서 Control + V (Windows) 또는 Command + V (Mac)를 눌러

붙여 넣으면 카드가 목록에 복사됩니다. 다른 보드 사이에서 작동합니다.

### ****■**카드 복사/ 이동**

카드 위로 마우스를 가져 가면 Control + X (Windows) 또는 Command + X (Mac)를 누르면

카드가 클립 보드에 복사됩니다. 

목록 위로 마우스를 가져 가면서 Control + V (Windows) 또는 Command + V (Mac)를 눌러

붙여 넣으면 카드가 목록으로 이동합니다. 다른 보드 사이에서 작동합니다.

### ****■**메뉴 닫기 / 편집 취소  **\- 트렐로 단축키****

> **Esc**

"esc"를 누르면 열린 대화 상자 창이 열리거나 편집중인 주석 및 주석이 팝업 또는 취소됩니다.

### **여기까지가 자주쓰는 단축키**

**아래는 알아두면 숙련자를 위한 단축키**

---

### ****■**\[검색 (검색어 입력 창으로 이동)\] - 트렐로 단축키**

> /

"/"를 누르면 커서가 헤더의 검색 상자에 놓입니다.

### ****■**마감일 설정하기 - 트렐로 단축키**

> **D**

"d"를 누르면 카드의 기간을 설정할 수 있는 달력이 열립니다.

### ****■**체크 리스트 추가 하기 - 트렐로 단축키**

> **\-**

“-”를 누르면 체크리스트가 카드에 추가됩니다.

### ****■**빠른 편집 모드 열기  **\- 트렐로 단축키****

> **E**

카드 위로 마우스를 가져가서 단축키 "e"를 누르면

빠른 편집 모드가 열리고 제목 및 기타 카드 속성을 빠르게 편집 할 수 있습니다.

### ****■**텍스트 저장  \- 트렐로 단축키**

> **Control + Enter (Windows)/ Command + Enter (Mac)**

를 누르면 작성중인 텍스트가 저장됩니다. 

주석을 쓰거나 편집 할 때, 카드 제목, 목록 제목, 설명 및 기타 사항을 편집 할 때 사용

### ****■**카드 필터 메뉴 열기**

> **F**

"f"를 사용하여 카드 필터 메뉴를 엽니 다. 제목 입력으로 검색이 자동으로 초점이 맞춰집니다.

### ****■**라벨 설정하기**

> **L**

"L"을 누르면 사용 가능한 라벨이 팝업됩니다. 

라벨을 클릭하면 카드에서 라벨을 추가하거나 제거합니다.

### ****■**라벨 이름 전환**

> **;**

세미콜론 키 ";"를 누름 보드의 라벨 이름을 표시하거나 숨깁니다. 

보드의 라벨을 클릭하여 전환 할 수도 있습니다.

### ****■**멤버 추가 / 제거**

> **M**

"m"을 누르면 멤버 추가 / 제거 메뉴가 열립니다. 

회원의 아바타를 클릭하면 그 사람을 할당하거나 할당을 해제합니다.

### ****■**나에게 할당 된 카드 만 보기**

> **Q**

"q"키를 누르면 "나에게 할당 된 카드"필터가 토글됩니다. 

(한던 버 누르면 다시 전체 보임)

### ****■**나에게 카드 할당하기**

> **스페이스 키**

"space"를 누르면 카드에 자신을 할당 (또는 할당 해제)합니다.
