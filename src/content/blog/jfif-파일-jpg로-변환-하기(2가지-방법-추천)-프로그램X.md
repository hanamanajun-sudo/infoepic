## **JFIF 파일이란?**

JFIF는 JPEG File Interchange Format의 약자로, 사실 JPG와 거의 같은 포맷이다. 확장자만 다를 뿐 이미지 압축 방식은 동일하다. 일부 웹사이트나 앱에서 이미지를 저장할 때 자동으로 .jfif 확장자로 저장되는 경우가 있어서 나중에 열리지 않는 상황이 생긴다.

## **방법 2: 그림판 이용**

1. .jfif 파일을 마우스 오른쪽 버튼 클릭
2. **연결 프로그램** → **그림판** 선택
3. 그림판에서 **파일 → 다른 이름으로 저장 → JPEG** 선택
4. 원하는 위치에 .jpg로 저장

그림판은 윈도우에 기본 설치되어 있어서 별도 프로그램이 필요 없다.

## **온라인 변환 사이트 이용**

인터넷에서 "jfif to jpg" 또는 "jfif 변환"으로 검색하면 무료 변환 사이트들이 나온다. 변환 과정에서 파일을 업로드해야 하므로 개인 사진은 로컬 방법(파일명 변경, 그림판)을 쓰는 게 더 안전하다.

**주의사항**: JFIF에서 JPG로 변환해도 화질 차이는 거의 없다. 단순히 확장자를 바꾸는 것이기 때문이다.

결론적으로 JFIF 파일은 확장자만 .jpg로 바꾸거나 그림판으로 다른 이름으로 저장하면 대부분 해결된다. 별도 프로그램을 설치할 필요가 없고 윈도우 기본 기능만으로 처리 가능하기 때문에 가장 빠른 방법이다. 변환 후에도 화질은 그대로 유지된다.
앞으로 비슷한 상황이 생기면 당황하지 말고 파일명 변경부터 시도해보자. 그것으로 해결이 안 되면 그림판을 쓰면 된다.
