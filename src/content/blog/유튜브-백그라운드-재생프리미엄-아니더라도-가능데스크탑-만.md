유튜브 백그라운드에서 재생 가능한데 여태 모르고 있었네요. 

여러분도 혹시 모르고 있다면 **프리미엄 아니더라도**

**데스크탑이라면** 자기가 하고 싶은 작업 하면서 유튜브 볼 수 있어요. 

너무 간단해서 외울 필요도 없습니다. 

## **유튜브 백그라운드 재생 방법**

보려는 유튜브 화면에 마우스 오른쪽 마우스 한번하도 또 한번 하세요 (마우스 오른쪽 클릭 두번)

**1\. 한번 오른쪽 누르면 이런거 나와요.**

![유튜브 백그라운드 재생(프리미엄 아니더라도 가능)데스크탑 만 이미지 1](/images/유튜브-백그라운드-재생프리미엄-아니더라도-가능데스크탑-만/img.png)

> \* 추가 팁으로 유튜브 영상 연속해서 듣고 싶은 노래 있으면 연속 재생 선택하면 되요.

**2\. 화면에 두번째 마우스 오른쪽 클릭 하면 이런거 나와요.**

![유튜브 백그라운드 재생(프리미엄 아니더라도 가능)데스크탑 만 이미지 2](/images/유튜브-백그라운드-재생프리미엄-아니더라도-가능데스크탑-만/img_1.png)

**3\. 여기서 PIP 모드 누르면 백그라운드 재생 됩니다.**

작업 하면서 유튜브 화면 떠 있어요. 

> \*PIP모드란?   
> 다른 작업을 하는 동시에 작은 화면으로 영상과 작업을 하는 기능

**4\. 유튜브 백그라운드 설정 가능**

**\- 화면 창도 이동가능**

 **-화면 크기 조절도 가능**

**\- 유튜브 백그라운드 상태에서 멈춤, 재생 가능**

**\- 탭돌아가기/ 화면 닫기 가능**

이 기능 모르셨다면 뉴월드 시작 입니다. 

유튜브 프리미엄 없어도 백그라운드 기능 사용하세요. 

이것 아니면

**유투브 광고제거, 백그라운드 재생 기능**이있는

유튜브 프리미엄을 쓰는 것도 방법입니다. 대신 이것 한번 사용하면 다시 못돌아감
