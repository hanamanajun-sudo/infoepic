컴퓨터로 글을 작성하다가 가끔가다가 글씨마다 스페이스 눌러서 쓴것 처럼  
자음과 모음이 분리되서 써지는 경우가 발생한다.

그럴때 별다른 방법을 찾지 못하고 pc를 재부팅하면 원래대로 돌아와서 그냥 그렇게 해결하곤 했는데

컴퓨터를 끄기 곤란한 경우가 생겨서 어떻게 해결할지 한참 고민하다가 발견한 방법을 소개하려고 한다.

\* 윈도우 H 단축키를 누르면 위와 같이 자음 모음이 분리되서 써지니까 주의하자. (종종 커뮤니티에서 낙시하는 장난)

## **자음모음 분리 해결 방법**

1.  윈도우 작업표시줄(화면 하단 아이콘 모여있는 바)에서 오른쪽 마우스 클릭  
      
    
2.  터치키보드 단추표시 선택  
    
    ![자음 모음 분리 3초 해결 방법 (윈도우+H 단축키 끄는 방법) 이미지 2](/images/자음-모음-분리-3초-해결-방법-윈도우H-단축키-끄는-방법/이미지 010.png)
    
      
      
    
3.  윈도우 작업표시줄 하단 오른쪽에 생긴 키보드 클릭  
    
    ![자음 모음 분리 3초 해결 방법 (윈도우+H 단축키 끄는 방법) 이미지 3](/images/자음-모음-분리-3초-해결-방법-윈도우H-단축키-끄는-방법/이미지 011.png)
    
4.  아무키나 누르고 키보드 빠져나오기  
    
    ![자음 모음 분리 3초 해결 방법 (윈도우+H 단축키 끄는 방법) 이미지 4](/images/자음-모음-분리-3초-해결-방법-윈도우H-단축키-끄는-방법/이미지 009.png)
    
5.  정상으로 돌아온 것 확인

해결이 되었다면 터치키보드는 원래대로 작업표시줄에서 지우면 되겠다.

해결이 되었다면 다른분들과 공유를 위해 하단에 좋아요나 댓글 남겨주세요~
