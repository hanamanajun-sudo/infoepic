아이폰 사파리 브라우저에서 보여지기 싫은 방문했던 페이지 혹은 방문 페이지를 근거로 자동 완성되는 주소가 싫어서 삭제를 원하시는 분들은 참고하세요 .

1\. 사파리 아이콘 선택 

![아이폰 사파리 방문기록 지우기 ( 깔끔 전체 삭제 방법 설명) 이미지 1](/images/아이폰-사파리-방문기록-지우기-깔끔-전체-삭제-방법-설명/img.jpg)

#### 2\. 사파리 들어가면 하단에 **책 모양 아이콘**을 누른다.

![아이폰 사파리 방문기록 지우기 ( 깔끔 전체 삭제 방법 설명) 이미지 2](/images/아이폰-사파리-방문기록-지우기-깔끔-전체-삭제-방법-설명/img_1.jpg)

#### 3\. 화면 상단의 3번째 **'방문기록'** 버튼 누른다. (왼쪽부터  책갈피, 읽기 목록, 방문기록)

![아이폰 사파리 방문기록 지우기 ( 깔끔 전체 삭제 방법 설명) 이미지 3](/images/아이폰-사파리-방문기록-지우기-깔끔-전체-삭제-방법-설명/img_2.jpg)

#### 4\. 방문기록 화면에서 하단의 지우기 선택

![아이폰 사파리 방문기록 지우기 ( 깔끔 전체 삭제 방법 설명) 이미지 4](/images/아이폰-사파리-방문기록-지우기-깔끔-전체-삭제-방법-설명/img_3.jpg)

#### 5\. 방문기록 지우기 기간 선택

지우기를 선택하면 지우려는 기간을 선택한다. 

전체, 오늘 및 어제, 오늘, 지난 1시간 중 선택

이제 열어본 페이지 방문기록, 쿠키 및 다른 탐색 데이터가 제거 된다.

![아이폰 사파리 방문기록 지우기 ( 깔끔 전체 삭제 방법 설명) 이미지 5](/images/아이폰-사파리-방문기록-지우기-깔끔-전체-삭제-방법-설명/img_4.jpg)

#### 6\. 기간 선택하면 바로 지워준다. 

![아이폰 사파리 방문기록 지우기 ( 깔끔 전체 삭제 방법 설명) 이미지 6](/images/아이폰-사파리-방문기록-지우기-깔끔-전체-삭제-방법-설명/img_5.jpg)

#### 7\. 테스트해본다.

이제 사파리 주소창에 검색을 할 때에 원치않는 자동완성 주소 등은 삭제되어 보이지 않는다. 

![아이폰 사파리 방문기록 지우기 ( 깔끔 전체 삭제 방법 설명) 이미지 7](/images/아이폰-사파리-방문기록-지우기-깔끔-전체-삭제-방법-설명/img_6.jpg)

iCloud로 여러 기기를 동기화하고 있다면 아이패드나 맥에 같은 방문기록이 남아있을 수 있습니다. 완전히 지우려면 해당 기기에서도 동일하게 삭제해야 해요. 아예 기록이 쌓이지 않게 하려면 사파리 하단의 탭 아이콘을 눌러서 **개인정보 보호** 모드로 전환하면 됩니다. 이 모드에서는 탭을 닫는 순간 기록이 남지 않아요.
