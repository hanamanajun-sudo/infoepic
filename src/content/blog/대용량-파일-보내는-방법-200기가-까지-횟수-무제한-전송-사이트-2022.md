일본 사이트인데 일본에서는 보편적으로 사용하고 있는 대용량 파일 전송 사이트라고 합니다. 

핸드폰에서도 사용 가능하고 드래그만 하면 업로드 주소가 생성되는데

언어를 모르더라도 사용 방법만 알고 있으면 쉽게 전송(업로드, 다운로드 가능) 할 수 있어요.

![대용량 파일 보내는 방법 (횟수 무제한 전송 사이트)-2022 이미지 1](/images/대용량-파일-보내는-방법-200기가-까지-횟수-무제한-전송-사이트-2022/제목을 입력해주세요_-001 (20) (1).png)

저는 주로 핸드폰에서 컴퓨터로 파일 보낼 때 사용하거나 용량 큰 파일 usb대신 사용하기도 합니다.

가입할 필요 없고 무료니까 선택할지 말지는 지금까지 공유하던 곳과 비교해 보세요.

## ****✔** 대용량 파일 전송 사이트 '기가파일' 장점**

-   무료
-   개수 무제한 
-   1개 파일에 200기가 까지 가능
-   파일 업로드 다운로드 회원 가입 없음 / 로그인 필요 없음

## **✔ 대용량 파일 전송 사이트 이용방법 (파일 업로드/ 다운로드)**

대용량 파일 보내는 방법

**1\. 기가파일(일본) 사이트에 접속한다.** 

[https://gigafile.nu/](https://gigafile.nu/)

![대용량 파일 보내는 방법 (횟수 무제한 전송 사이트)-2022 이미지 2](/images/대용량-파일-보내는-방법-200기가-까지-횟수-무제한-전송-사이트-2022/이미지 001.png)

\* 위와 같은 사이트가 표시되는데 무시하고 스크롤을 내린다.

**2\. 업로드 할 파일의 보관 기간을 선택한다.**  

**화면 상단의 7일, 14일, 21일, 30일, 60일 중에 하나를 선택**

\* 업로드 한 파일의 선택 기간이 지나면 자동으로 삭제

![대용량 파일 보내는 방법 (횟수 무제한 전송 사이트)-2022 이미지 3](/images/대용량-파일-보내는-방법-200기가-까지-횟수-무제한-전송-사이트-2022/이미지 003.png)

**3\. 업로드할 파일을 선택한다.** 

**①에  드로그앤 드롭을 해서 업로드** 할 파일을 날짜 밑에 옮겨도 되고 

②에 파일 선택 버튼을 눌러서 파일을 선택 할 수도 있다. 

**(+핸드폰에서는 ②번 ファイルを選択- 눌러서 불러오면 된다.  아이폰도 가능)**

③파일명 입력 가능 (선택사항)

④다운로드 암호를 설정 가능(선택사항) 

**4\. 여러 파일을 업로드 할 때에는 ⑤버튼을 탭하여 하나의 파일(zip)로 묶기 선택 가능**

\* 묶지 않으면 상대방이 하나하나 다운로드 받아야 함

**5\. 업로드 완료**

![대용량 파일 보내는 방법 (횟수 무제한 전송 사이트)-2022 이미지 4](/images/대용량-파일-보내는-방법-200기가-까지-횟수-무제한-전송-사이트-2022/이미지 005.png)

생성된 주소를 줄이고 싶으면 위 사진의 ②클릭을 하자

주소 줄이는 방법 

![대용량 파일 보내는 방법 (횟수 무제한 전송 사이트)-2022 이미지 5](/images/대용량-파일-보내는-방법-200기가-까지-횟수-무제한-전송-사이트-2022/이미지 010.png)

위 사진 처럼 별도 페이지 나와서 짧게 하기 버튼 ① 클릭

![대용량 파일 보내는 방법 (횟수 무제한 전송 사이트)-2022 이미지 6](/images/대용량-파일-보내는-방법-200기가-까지-횟수-무제한-전송-사이트-2022/이미지 009.png)

위 사진의 ①번이 짧아진 주소

**6.  표시된 주소 ①을 상대에게 카톡이나 기타 SNS, 메일로 공유합니다.**

복잡해 보이지만 직접 해보면 너무 간단하다.

**정리하자면**

> **파일 선택해서 업로드 >  주소 복사 > 끝  
> (파일이 여러개면 오렌지색 버튼을 클릭한다. )**

> 생성된 주소만 메일이나 SNS로 공유하면 알아서 파일 링크를 통해 다운로드가 가능한 점도 마음에 들고  
> 100기가도 아니고 200기가 까지 무료라서 넉넉하게 여유있게 공유가 가능합니다. 

## **대용량 파일 받는 방법**

1\. 파일을 보내는 사람으로 부터 받은 주소로 들어간다. 

![대용량 파일 보내는 방법 (횟수 무제한 전송 사이트)-2022 이미지 7](/images/대용량-파일-보내는-방법-200기가-까지-횟수-무제한-전송-사이트-2022/이미지 006.png)

① 한꺼번에 다운로드 

② 웹에서 파일 확인하기 페이지 (まとめる파일은 다운로드 하기 전에 웹에서 내용을 확인할 수 있다. )

③ 개별 파일 다운로드

④ 파일 삭제하기 (상단에서 삭제 키를 기억해야 한다)

용량 제한이나 갯수 제한 등이 있어서 좀더 사용하기 편한 파일 전송 방법 입니다.

사용하시면 좋을듯 싶은데 한번 사용해 보고 선택하면 됩니다. 

**보안에 크게 문제는 없을 듯 싶지만 그래도 신경 쓰인다면**

**다른 사이트 이용하거나 보안 상관 없는 파일만 공유하는 걸 추천합니다.**
