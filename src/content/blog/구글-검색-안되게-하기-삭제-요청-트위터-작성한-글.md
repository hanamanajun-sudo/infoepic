트윗이나 본인이 작성한 글을 구글에서 검색 안되게 하는 방법입니다. 

구글에서 검색이 안되게 하려면 일단 먼저 본인이 작성한 글을 삭제 합니다. 

![구글 검색 안되게 하기 (삭제 요청) -트위터, 작성한 글 이미지 1](/images/구글-검색-안되게-하기-삭제-요청-트위터-작성한-글/img.png)

내가 작성한 글이 아닌데 타인의 게시물로 인해 내 권리가 침해 되었거나

내 정보를 노출하거라면 사이트에 개인정보 삭제 요청을 하세요.

사이트에 요청이 어려운 상황이라면 구글에 삭제 요청을 할 수도 있습니다. 

삭제를 하더라도 구글이 인덱싱한 경우에는 오랫동안 검색에 남아있는 경우가 있습니다.

이럴 때에는 구글에 삭제 요청을 해야 합니다. 

## **Google 검색에서 오래된 콘텐츠 삭제 요청 페이지**

이미 수정된 내용인데 검색 되거나 삭제된 페이지 삭제된 이미지인데 검색이 되는 경우에

요청을 하시면 됩니다.  

**[https://t.co/4vnYK5XDND?amp=1](https://t.co/4vnYK5XDND?amp=1)**

### **개인정보 혹은 법적인 문제가 있는 콘텐츠 삭제 요청의 경우**

법을 위반하거나 권리를 침해한다고 생각하는 콘텐츠의 경우는 아래를 구글 공식 페이지를 참고하세요

[support.google.com/legal/answer/3110420?visit\_id=637541620620860812-2158706583&rd=1](https://support.google.com/legal/answer/3110420?visit_id=637541620620860812-2158706583&rd=1)

실제로 삭제 요청을 넣으면 구글에서 처리하는 데 보통 며칠에서 길게는 수 주가 걸립니다. 처리 상태는 구글 서치콘솔의 삭제 탭에서 확인 가능해요. 트위터 글이라면 계정을 비공개로 전환하는 게 더 빠른 응급처치가 될 수 있습니다. 삭제 요청은 원본 페이지가 이미 사라진 상태일 때 더 빠르게 처리됩니다.
