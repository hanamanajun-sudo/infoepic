언젠가 부터 유튜브에 시간 부분이 남은 시간(마이너스로 시간표시)으로

재생시간이 표시 되는 문제가 있어서 왜그런가 바꾸는 방법이 없나

찾아보다가 알게 된 사실이 있어서 같은 문제를 겪는 답답해 할 분들을 위해

바꾸는 방법 공유 합니다. 

## **유튜브 재생시간 마이너스 표시 바꾸는 방법**

유튜브 시간 표시 부분이 보통 플레이 된 시간을 표시 하는게 기본 설정 인데

이것을 본 시간이 아닌 남은 시간으로 바꾸는 방법이 있습니다.

**시간 부분을 클릭하면 됩니다** 

![유튜브 재생시간 마이너스 표시 바꾸는 방법(남은 시간 ↔ 시청 시간) 이미지 2](/images/유튜브-재생시간-마이너스-표시-바꾸는-방법남은-시간-↔-시청-시간/blob_1.jpg)

**그러면 경과시간에서 영상의 남은 시간으로 표시가 바뀝니다.** 

![유튜브 재생시간 마이너스 표시 바꾸는 방법(남은 시간 ↔ 시청 시간) 이미지 3](/images/유튜브-재생시간-마이너스-표시-바꾸는-방법남은-시간-↔-시청-시간/blob_2.jpg)

**그러니까 현재 마이너스 표시(남은시간)가 되어 있다면 본인도 모르게**

**시간 부분을 터치했다는 것이겠죠**
