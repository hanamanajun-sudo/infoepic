유튜브 보다보면 자막 크기가 너무 작다고 생각 할 때가 있는데 이것 크기 조절하는 

기능이 있는 걸 처음 알았어요. (충격)

혹시 저처럼 모르는 사람 있으면 충격 도움 받으시라고 꿀팁 공유 드립니다. 

아 그 외에도 자막 관련 단축키 기능을 찾아서 모았어요.

다는 필요 없고 자신이 필요한 유튜브 이용 시 단축키 기능 만 외어 두세요.

---

### **▶유튜브 자막 보기 단축키**

**C**

단축키 C 누르면 자막이 켜진다. 한번더 누르면 사용 중지 전환

---

### ****▶**유튜브 자막 글꼴 크기 크게 키우기**

자막 켜진 상태에서 

**+**

단축키 + 를 누르면 자막이 커진다. 

---

### ****▶**유튜브 자막 글꼴 크기 작게 줄이기**

자막 켜진 상태에서 

**\-**

단축키 -를 누르면 자막이 작아진다.

### 자동 번역 자막 켜는 방법

자막이 없는 외국어 영상에서도 자동 번역 자막을 쓸 수 있다.

설정(톱니바퀴) 클릭 > 자막 > 자동 번역 > 언어 선택

다만 자동 번역이라 번역 품질이 완벽하진 않다. 영어 영상을 한국어로 볼 때 제일 무난하게 나오는 편.

### 자막 스타일 변경

유튜브 설정에서 자막 글꼴, 배경색, 글자색을 바꿀 수 있다.

계정 설정 > 자막 에서 변경 가능하고 한번 설정해두면 모든 영상에 일괄 적용된다.

### 단축키가 안 들어올 때

전체화면이나 시어터 모드에서는 단축키가 잘 먹히는데 일반 창 모드에서는 안 되는 경우가 있다. 그리고 YouTube Premium 없어도 자막 기능은 다 무료로 쓸 수 있다.

영상마다 자막이 없는 경우도 있고, 자동 생성 자막은 정확도가 좀 낮다는 거 참고.

## **유튜브 자막 크기·불투명도 조절 단축키 전체**

| 단축키 | 기능 |
|--------|------|
| C | 자막 켜기/끄기 |
| + (자막 켜진 상태) | 자막 크기 키우기 |
| - (자막 켜진 상태) | 자막 크기 줄이기 |
| O | 자막 불투명도 줄이기 |
| W | 자막 배경창 불투명도 줄이기 |

이 단축키들은 유튜브 자체 단축키이므로 유튜브 영상 재생 화면에서만 작동한다. 다른 창에 포커스가 있으면 작동하지 않으니 영상 화면을 먼저 클릭해두자.

## **자막 상세 설정 방법 (고급)**

단축키 외에 더 세밀하게 조절하려면 유튜브 설정에서 변경 가능하다.

1. 영상 하단의 자막(CC) 버튼 오른쪽 **설정(톱니바퀴)** 클릭
2. **자막/CC** 선택
3. **옵션** 클릭 → 글꼴, 크기, 색상, 배경, 윤곽선 등 세부 조정 가능

여기서 변경한 설정은 유튜브 계정에 저장되어 다른 영상에서도 유지된다. 눈이 좋지 않거나 화면이 작은 환경에서 볼 때 크게 설정해두면 편하다.

자막 단축키 중 가장 많이 쓰이는 건 역시 C 키다. 자막이 없는 영상인데 켜놨거나, 자막이 자꾸 방해될 때 빠르게 끄고 켤 수 있어서 유용하다. 크기 조절(+/-)은 나이 드신 분들이나 작은 화면에서 보는 분들이 한 번 설정해두면 두고두고 편하게 쓸 수 있다. 자막 설정은 유튜브 계정에 저장되기 때문에 한 번만 맞춰두면 다음에도 유지된다.
자막 단축키는 유튜브를 자주 보는 사람이라면 한 번 익혀두는 것이 좋다. 특히 청각 장애가 있거나 소음이 심한 환경에서 자막을 활용할 때 크기와 불투명도 조절이 매우 유용하다.
자막 기능은 청각이 불편한 분들뿐 아니라 주변 소음이 심할 때, 조용히 영상을 봐야 할 때도 매우 유용하다. 단축키 하나만 외워도 훨씬 편한 시청 경험이 가능하다. 특히 C 키 하나만으로 자막 온오프가 가능하다는 것을 모르는 사람이 의외로 많다.
